# 飞机大战网页版游戏

一款基于HTML5 Canvas和JavaScript开发的2D纵向卷轴射击游戏，采用现代化的设计模式和架构。

## 🚀 游戏特色

### 核心玩法
- **经典射击体验**: 2D纵向卷轴射击，操控战机消灭敌人
- **武器升级系统**: 5级武器升级，从单发到激光武器
- **特殊技能**: 全屏炸弹清除所有敌人
- **护盾系统**: 临时无敌保护
- **生命系统**: 3条生命，支持道具回血

### 游戏系统
- **多种敌机**: 不同类型的敌机，各具特色
- **道具系统**: 武器升级、护盾、生命恢复等道具
- **分数排行榜**: 本地存储的分数记录
- **成就系统**: 多种成就解锁
- **音效支持**: 背景音乐和音效反馈

### 视觉效果
- **粒子系统**: 爆炸、推进器等特效
- **动态背景**: 星空滚动效果
- **现代UI**: 科幻风格的用户界面
- **平滑动画**: 60FPS游戏体验

## 🎮 操作方式

### 键盘控制
- **WASD** 或 **方向键**: 移动战机
- **空格键**: 射击
- **F键** 或 **X键**: 使用特殊技能
- **ESC键** 或 **P键**: 暂停/继续游戏
- **Enter键**: 确认
- **ESC键**: 取消/返回

### 鼠标控制
- 支持鼠标移动控制（可在设置中切换）
- 左键射击

### 触摸控制
- 支持移动设备触摸操作

## 🏗️ 技术架构

### 设计模式
- **单例模式**: 管理器类（输入、音频、状态）
- **对象池模式**: 优化内存使用，减少GC
- **状态模式**: 游戏状态管理
- **工厂模式**: 游戏对象创建
- **观察者模式**: 事件系统

### 核心系统
- **输入管理器**: 统一处理键盘、鼠标、触摸输入
- **音频管理器**: 音效和背景音乐管理
- **状态管理器**: 游戏状态切换
- **粒子系统**: 特效和动画
- **对象池管理**: 性能优化
- **碰撞检测**: 高效的碰撞检测系统

## 🛠️ 开发环境

### 系统要求
- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 支持HTML5 Canvas
- 支持ES6+语法
- 本地服务器（推荐）

### 运行方式
1. **直接打开**: 双击`index.html`文件
2. **本地服务器**: 使用Live Server或其他本地服务器
3. **Python服务器**: `python -m http.server 8000`
4. **Node.js服务器**: `npx serve .`

### 开发工具
- **F12键**: 切换调试信息显示（开发环境）
- **浏览器开发者工具**: 查看控制台日志
- **性能监控**: FPS显示和对象计数

## 🎨 游戏设计

### 战机类型（规划中）
- **均衡型**: 标准的火力、速度和防御
- **速度型**: 高速移动但火力较弱
- **重型**: 强大火力但移动缓慢

### 敌机类型
- **侦察机**: 低生命值，直线飞行
- **战斗机**: 中等生命值，会发射子弹
- **轰炸机**: 高生命值，投掷炸弹
- **精英敌机**: 特殊能力和攻击模式
- **BOSS**: 多阶段战斗，需要攻击弱点

### 道具类型
- **武器升级**: 提升武器等级
- **生命恢复**: 回复生命值
- **护盾**: 临时无敌状态
- **炸弹**: 全屏清敌
- **分数加成**: 双倍分数
- **僚机**: 辅助射击

## 🏆 成就系统

### 基础成就
- **首次击杀**: 击落第一架敌机
- **初出茅庐**: 获得1000分
- **小有名气**: 获得10000分
- **完美操作**: 无伤通过一关
- **BOSS杀手**: 击败一个BOSS

### 进阶成就
- **连击大师**: 连续击杀50架敌机
- **收集达人**: 收集所有道具类型
- **生存专家**: 单局存活10分钟
- **武器大师**: 达到最高武器等级
